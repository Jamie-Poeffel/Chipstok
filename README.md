## 📥 Installation

1. Clone the repository:
   ```sh
   git clone https://github.com/Jamie-Poeffel/chipstok.git
   ```
2. Navigate to the project folder:
   ```sh
   cd chipstok
   ```
3. Install dependencies:
   ```sh
   npm install
   ```
4. Run the application:
   ```sh
   npm start
   ```
